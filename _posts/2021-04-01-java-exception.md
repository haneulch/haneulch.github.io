## Checked Exception

* Runtime Exception을 상속하지 않음
* Compile 시점의 Exception
* 반드시 예외 처리
> IO Excetion, ClassNotFoundException ...

## Unchecked Exception

* 프로그래밍적 실수
> NullPointerException, IndexOutOfBoundException
